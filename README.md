# 📹 Video Calling Application

## 🚀 Overview
This project is a **real-time video calling web application** developed using **Spring Boot (backend)** and **HTML, CSS, JavaScript (frontend)**. It integrates **ZEGOCLOUD UIKit Prebuilt** to enable seamless one-to-one video communication.

---

## ✨ Features
- 🔐 User Registration & Login
- 👤 User Management (Create & Store Users)
- 📞 One-to-One Video Calling
- 🎥 Real-time Audio & Video Streaming
- 🌐 Browser-based (no installation required)
- 💡 Simple and responsive UI

---

## 🛠️ Tech Stack

### Backend
- Java
- Spring Boot
- REST APIs

### Frontend
- HTML
- CSS
- JavaScript

### Video Integration
- ZEGOCLOUD UIKit Prebuilt
